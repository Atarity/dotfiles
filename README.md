# Dotfiles

| App | Config | Path |
| -- | -------- | ---- |
| zsh | `.zshrc` | `~/` |
| Bash | `.bash_profile` | `~/` |
| Tmux | `.tmux.conf` | `~/` |
| iTerm2 | `Afterglow.iterm` | Pref.→ Profiles → Colors → Import |
| MacOS Terminal | `Afterglow.terminal` | Pref.→ Profiles → Import |
| Htop | `htoprc` | `~/.config/htop/` |
| Ranger | `rc.conf` | `~/.config/ranger/` |
